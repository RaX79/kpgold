Тестовое задание для АО "Ксеньевский Прииск"

Содержание тестового задания:
Создать список пользователей (Поля ФИО, Дата рождения, Пол, Подразделение).

Предусмотреть редактирование, удаление и добавление новых на сайте используя php, mysql, javascript (ajax), html (без использования фреймворков).

Информация о программном обеспечении, при помощи которого выполнено тестовое задание:
PHP Version 5.3.13
5.5.25 - MySQL Community Server (GPL)
Полная информация о PHP находится на странице phpinfo().htm

Назначение каждого файла обозначено в комментарии.

Кодировка файлов utf-8.
В тестовом задании не используется .htaccess, поэтому объявление кодировки явно указано в файле head.php.
Не используется cookie.
Для передачи сообщений пользователю об успехе операций или ошибках используется суперглобальный массив session.
В задании не используется jquery, так как является фреймворком.

Подключение к базе данных не закрывается умышленно, поскольку после выполнения скрипта закрытие происходит автоматически и в данном случае является излишним.
